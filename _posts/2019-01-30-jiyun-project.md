# 게시판 만들기 프로젝트

**프로젝트 개요**
-사용언어: Java 1.8
-프레임워크: Spring 3.9.3
-DB: MySQL 8.0
-서버 : Apache Tomcat 8.5
-빌드: Maven
-테스트: JUnit 4.12 
-ORM: Mybatis 3.4.6

**환경 설정**
-Maven dependencies 추가 (pom.xml에 dependencies 추가)
-MyBatis로 MySQL 연동(root-context.xml 파일에 beans 추가), XML Mapper에 SQL 저장
